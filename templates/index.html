Bienvenido a la carga de datos

Nuestra plataforma utiliza una API desarrollada en Django y Django Rest Framework (DRF) como motor principal para el procesamiento de datos. Esta API facilita la visualización y exploración de información sobre establecimientos educativos y productivos. Integrada con PostgreSQL y la extensión PostGIS, gestiona eficientemente datos geoespaciales.

Por favor, dirígete a la sección "Importar" para cargar archivos en formato XLSX o CSV y poblar las tablas.
